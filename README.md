jquery-loading
==============

Emaple
--------------

Show Loading

    $("#id").showLoading();

Hide Loading

    $("#id").hideLoading();

Show Loading With Overlay

    $("#id").showLoading({ full:true });

Show Loading With Animation

    $("#id").showLoading({ animate:true, speed:400 });

Show Loading With Delay

    $("#id").showLoading({ delay:400 });
